## Architecture
- **Backend** (FastAPI / Python)
  - `/api/health`
  - `/api/mindmap/parse-pdf` — free local heuristic outline (pypdf)
  - `/api/mindmap/from-pdf` — AI mode (Claude Sonnet 4.5, session-gated, BYO-key bypasses counter)
  - `/api/auth/*` — Emergent Google OAuth + session cookies
  - `/api/billing/create-checkout`, `/api/billing/portal`, `/api/webhook/stripe`
  - MongoDB: users, sessions, subscriptions, usage_limits
- **Frontend** (React 19 + Tailwind + Sora font)
  - `/` — Cosmic landing (cinematic teaser, features, CTA)
  - `/app` — Studio (sidebar, canvas, context menu, PDF dialog, upgrade dialog)
  - Access gate + Auth context + local-first mind-map storage
